From frontend/src/app/routes.cljs:
```clojure
(def router
  (reitit-fe/router
    [["/" {:name :page/home}]
     ["/inscription" {:name :page/signup}]
     ...]))

(defn init! []
  (reitit-fe-easy/start!
    router
    (fn [match _]
      (let [page (or (-> match :data :name) :page/home)]
        (rf/dispatch [:router/navigated page])))
    {:use-fragment false}))
```

From frontend/src/app/events.cljs:
```clojure
(rf/reg-event-db :router/navigated
  (fn [db [_ page-name]]
    (cond-> (assoc db :router/current-page page-name)
      (not= page-name :page/signup) (dissoc :eligibility/join-network))))
```

From frontend/src/app/views.cljs:

(defn current-page []
