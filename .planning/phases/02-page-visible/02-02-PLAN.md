**Sub-components to create (all in same file):**

1. **`energy-type-label`** (private helper): Maps keyword to French label. `{:solar "Solaire" :wind "Eolien" :hydro "Hydro" :biomass "Biomasse"}` with `"Autre"` as default.

2. **`energy-mix-colors`** (private def): `{:solar "#f5aa46" :wind "#64917d" :hydro "#4a90d9" :biomass "#8b6914"}` with default `"#999999"`.

3. **`network-hero`** (private fn): Subscribes to `:network-detail/network` and `:network-detail/stats`. Renders:
   - `[:section.nd-hero]` with `[:h1.nd-title (:network/name network)]`
   - Generated description paragraph (CONT-02): "Rejoignez l'operation d'autoconsommation collective **[name]**. Ce reseau reunit [N] site(s) de production pour une capacite totale de **[X] kWc**."
   - Handle plural: "site" if 1, "sites" if >1

4. **`network-stats`** (private fn): Subscribes to `:network-detail/stats`. Renders `[:section.nd-stats]`:
   - Stat card for total capacity: value = `(:total-capacity-kwc stats)`, unit = "kWc", label = "Capacite installee" (STAT-01)
   - Stat card for production count: value = `(:production-count stats)`, label = "Sites de production"
   - Stat card for consumer count: value = `(:consumer-count stats)`, label = "Consommateurs"
   - Each card: `[:div.nd-stat-card [:span.nd-stat-value value] [:span.nd-stat-label label]]`

5. **`energy-mix-bar`** (private fn): Takes `energy-mix` map as arg. If empty map, renders "Aucune production active". Otherwise renders `[:div.nd-mix-bar]` with flex segments colored by energy type, each segment `[:div.nd-mix-segment {:style {:width (str pct "%") :background color}}]`. Below the bar, a legend: `[:div.nd-mix-legend]` with `[:span.nd-mix-legend-item]` showing colored dot + "Type X%" for each entry sorted by pct descending. (STAT-02)

6. **`network-map-section`** (private fn): Form-3 component using `r/create-class`. Subscribes to `:network-detail/network` in outer let.
   - Local atoms: `map-el`, `map-inst`, `circle` (single atom, not vector)
   - `component-did-mount`: Call `google-maps/load-google-maps-script!` with callback that creates `js/google.maps.Map` centered on network coords, zoom 13, mapTypeId "roadmap". Then call `google-maps/draw-circle!` with network's center-lat, center-lng, radius-km. Store result in circle atom. Call `.fitBounds` on the map with `(.getBounds @circle)`.
   - `component-will-unmount`: If `@circle`, call `(.setMap @circle nil)`.
   - `reagent-render`: Returns `[:section.nd-map-section [:h2 "Zone du reseau"] [:div.nd-map-container {:ref (fn [el] (when el (reset! map-el el))) :style {:width "100%" :aspect-ratio "4/3" :min-height "300px" :max-height "500px"}}]]`.
   - CRITICAL: The map div must ALWAYS be rendered (no conditional wrapping). The h2 is outside the map div. (MAP-01)

7. **`production-list`** (private fn): Subscribes to `:network-detail/productions` and `:network-detail/has-productions?`. Renders `[:section.nd-productions]`:
   - `[:h2 "Sites de production"]`
   - If no productions: `[:p.nd-empty "Aucun site de production pour le moment."]`
   - If productions: `[:div.nd-production-grid]` with one card per production:
     ```
     [:div.nd-production-card {:key (:production/id prod)}
       [:span.nd-production-type (energy-type-label (:production/energy-type prod))]
       [:span.nd-production-power (str (:production/installed-power prod) " kWc")]
       [:span.nd-production-address (:production/producer-address prod)]]
     ```
   - (CONT-01, MAP-02: productions listed below the map since no lat/lng coordinates available)

8. **`join-cta`** (private fn): Renders `[:section.nd-cta]`:
   - `[:h2 "Rejoignez ce reseau"]`
   - `[:p "Testez votre eligibilite et commencez a beneficier de l'energie locale."]`
   - `[:a.btn.btn--accent.nd-cta-btn {:href (rfee/href :page/home)} "Tester mon eligibilite"]`
   - CTA links to home page where eligibility form exists (per research recommendation). (CONT-03)

9. **`network-detail-page`** (public fn): Keep the loading?/error/data/else cond structure. Replace the `data` branch with:
   ```clojure
   [:div.nd-page.container
     [network-hero]
     [network-stats]
     [energy-mix-section]   ;; wraps energy-mix-bar with h2
     [network-map-section]
     [production-list]
     [join-cta]]
   ```
   Add an `energy-mix-section` wrapper: subscribes to `:network-detail/stats`, renders `[:section.nd-energy-mix [:h2 "Mix energetique"] [energy-mix-bar (:energy-mix stats)]]`.
